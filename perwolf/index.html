Perwolf: a noisy type design project
About Perwolf

This tool allows you to apply a Perlin noise texture to the outline of your text in the desired OpenType font and download the result as a Scalable Vector Graphics object for further use. Inspired by the randomfont FF Beowolf created by Erik van Blokland and Just van Rossum in 1990.

The javascript code uses the p5.js, opentype.min.js and p5.svg.min.js libraries. The default font as well as the display font is Fanwood Regular by Barry Schwartz.

Created by Matija Resman with the guidance of Prof. Pierre Pané-Farré and Simon Thiefes as a semester project for the Co-Op Monday course at HAW Hamburg in 2024.
